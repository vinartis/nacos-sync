# OpenSpec for nacos-sync

This directory contains OpenSpec change proposals and specifications for the nacos-sync project.

## Structure
- `project.md` - Project overview, scope, stack, capability map
- `changes/` - Change proposals (in progress)
- `specs/` - Approved specifications per capability

## Conventions
- Change names: `kebab-case`, imperative mood (e.g., `upgrade-nacos-client-to-3x`)
- Each change must have `proposal.md`; `design.md` / `tasks.md` / `specs/` are added as needed
- Specs are organized by capability as defined in `project.md`
